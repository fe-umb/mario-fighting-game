# mario-fighting-game
Mario ~fighting~ game in prolog =)

queries:  
maisforte(Personagem1,Personagem2) -> checks if character 1 is stronger than character 2  
derrotado(Heroi,Vilao) -> checks if the hero is able to defeat the villain  
combate(Heroi,Vilao,X,Y) -> returns x (hero's hp) and y (villain's hp) after one battle  

queries examples:  
1) ataque(X, 25).  
2) maisforte(spike, mario).  
3) derrotado(luigi, goomba).  
4) combate(peach, goomba, X, Y).
